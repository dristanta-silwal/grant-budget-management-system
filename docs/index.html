Grant Budget Management System

Overview

The Grant Budget Management System (GBMS) is a web-based platform designed to streamline grant budgeting for researchers and administrators. It includes dynamic budget calculation, role assignments, and Excel export features.

Live Demo

Features

  • Secure user authentication and role-based permissions.
  • Create, manage, and track grant budgets dynamically.
  • Export budgets as well-formatted Excel files.
  • Supports fringe and indirect cost calculations.
